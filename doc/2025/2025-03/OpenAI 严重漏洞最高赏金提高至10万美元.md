#  OpenAI 严重漏洞最高赏金提高至10万美元

**编译：代码卫士**  
  
**人工智能技术巨头OpenAI 将最高漏洞赏金从2万美元提高至10万美元，而这是该公司设立发现基础设施和产品中严重和高危漏洞外包计划的一部分。**  
  
  
![](https://mmbiz.qpic.cn/mmbiz_png/oBANLWYScMSa7dHABOx5QgzibDS0GN0n4KD9I0iayKicHEfvAOylNYUDTPGiaTg7enF0MKNkAUdDmJYedO98ohQKwA/640?wx_fmt=png&from=appmsg "")  
  
  
该漏洞奖励计划是OpenAI 公司设立的多项安全计划之一，这些安全计划包括资助安全研究项目、持续的竞争红队以及参与开源软件社区。除了提高严重漏洞的赏金外，OpenAI表示将为在有限时间内提交的高质量报告提供奖金。  
  
OpenAI 公司还宣布扩展网络安全资助计划。自2023年推出以来，该计划已经资助了28项研究计划。该公司表示受资助的项目已解决多种问题，如提示注入、安全代码生成以及自动化网络安全防御措施开发。该计划目前正邀请黑客提出软件打补丁、模型隐私、威胁检测和响应、安全集成以及复杂攻击弹性等的项目。OpenAI提到，该计划还以API积分的形式引入微资助，帮助研究人员快速做出创造性安全解决方案的模型。  
  
OpenAI 公司表示，正在与学术界、政府和商业实验室专家协作来对齐技能差距并改进其模型识别和修复漏洞的能力。该公司还与风投支持的创业公司 SpecterOps 合作，在企业、云和生产环境中开展可持续的竞争性红队活动。  
  
OpenAI 公司表示这些模拟攻击是为了赶在恶意人员利用之前找到潜在弱点。
