## 0x01 Jeecg_Tools工具介绍  
  
本工具为jeecg框架漏洞利用工具非jeecg-boot！  
  
包含poc：  
-  JEECG 登录绕过检测  
  
-  JEECG jeecgFormDemo文件上传  
  
-  JEECG common文件上传  
  
-  JEECG icon文件上传  
  
-  JEECG Xstream反序列化  
  
  
运行于jdk8环境  
  
```
java -jar Jeecg_Tools-1.0-java8.jar

```
