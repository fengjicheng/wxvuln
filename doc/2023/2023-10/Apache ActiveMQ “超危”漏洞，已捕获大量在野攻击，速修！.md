01 漏洞概况****  
  
  
  
Apache ActiveMQ 是美国阿帕奇（Apache）基金会的一套开源的消息中间件，它实现了 Java Message Service (JMS) 规范。作为一个消息中间件，它充当了应用程序之间的通信桥梁，允许不同的应用程序在分布式环境中进行可靠的异步通信。微步漏洞团队通过“X 漏洞奖励计划”获取到 Apache ActiveMQ 远程命令执行漏洞情报。攻击者可利用该漏洞执行任意命令，获取主机控制权限，建议受影响的用户做好自查及防护。  
**微步在捕获该漏洞情报时，已于第一时间（8月23日）通知订阅报告客户做好防护。该产品用量大且已捕获到攻击行为，建议升级新版本修复漏洞。******
