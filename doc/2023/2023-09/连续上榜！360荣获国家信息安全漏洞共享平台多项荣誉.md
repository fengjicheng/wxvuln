#  连续上榜！360荣获国家信息安全漏洞共享平台多项荣誉   
 360数字安全   2023-09-27 17:36  
  
****  
****  
近日，国家信息安全漏洞共享平台（简称CNVD）启动2022年度技术组支撑单位的能力评价工作，360被授予**“漏洞信息报送突出贡献单位”“CNVD协作特别贡献单位”**两项殊荣，并凭借优秀的  
**漏洞信息收集能力在年度考核中获得三星评级（最高级）**，“漏洞技术分析”“集体任务协作”两项也获高星评级。作为最早与CNVD合作的技术单位之一，360曾多次获评此类荣誉。  
  
  
![](https://mmbiz.qpic.cn/sz_mmbiz_png/GEQRwQFvibyIOyicbr8s1Upd7XtCEbAq3dMyH9Jia7FQAvEbyXl1Fdb9JXtGxvGXPg9xRMcLLlCOQHT4iaXibQKPXXA/640?wx_fmt=png "")  
  
  
![](https://mmbiz.qpic.cn/sz_mmbiz_png/GEQRwQFvibyIOyicbr8s1Upd7XtCEbAq3dXvmzo2YKRXfO09r7JNCia9Rtic3Ifx45gzyQuibPEABq3W46Cvp8JHSfw/640?wx_fmt=png "")  
  
  
国家信息安全漏洞共享平台（CNVD）是目前国内最权威的官方漏洞平台之一，此次表彰是对支撑单位在漏洞收集、漏洞发现、漏洞威胁风险大数据、漏洞技术分析、重大漏洞事件响应、集体任务协作等方面的工作进行了统计和评价。  
**近年来，360积极响应CNVD工作，此次获奖无疑是国家对360在漏洞统一收集验证、预警发布、建立应急处置体系等一系列工作的肯定。**  
  
   
  
漏洞风险关乎国家安全，做好漏洞风险防范治理是维护国家安全、保障网络空间稳定的必然要求。360近年来致力于网络安全威胁检测、分析与防御的研究与网络安全防护体系构建，  
**并依托360漏洞云积极开展在安全漏洞管理技术方面的探索与实践：**  
  
  
  
一是深入信息安全领域的监测与应急工作，实时监测网络空间的漏洞动态，为国家提供快速的监测与预警服务。  
  
  
二是围绕漏洞生态体系打造集漏洞监测、漏洞收集、漏洞挖掘、漏洞存储、漏洞管理、专家响应、安全服务定制化于一体的漏洞安全一站式服务。  
  
  
三是通过优质的漏洞情报订阅服务、可控的安全众测/众包服务、及时的漏洞应急响应服务、高效的漏洞补丁推送服务，以及专业的漏洞安全专家服务，为国家、政企客户抢占风险预警处置先机。  
  
  
**360已连续多年成为国家信息安全漏洞库（CNNVD）、网络安全威胁和漏洞信息共享平台（NVDB）和国家信息安全漏洞共享平台（CNVD）的技术支撑单位。**  
未来，360将全力配合各平台做好网络安全应急服务支撑工作，提升行业信息安全分析预警能力、为国家网络和信息安全保障工作贡献力量，全面守护国家网络安全！
