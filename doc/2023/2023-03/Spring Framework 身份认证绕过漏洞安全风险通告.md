**安全通告**  
  
  
  
Spring Framework 是一个开源应用框架，旨在降低应用程序开发的复杂度。  
它是轻量级、松散耦合的，具有分层体系结构，允许用户选择组件，同时还为 J2EE 应用程序开发提供了一个有凝聚力的框架。  
  
  
近日，奇安信CERT监测到官方发布了**Spring Framework 身份认证绕过漏洞(CVE-2023-20860)**，当Spring Security使用mvcRequest  
Matcher配置并将"**"作为匹配模式时，在Spring Security 和 Spring MVC 之间会发生模式不匹配，最终可能导致身份认证绕过。**鉴于此产品用量较大，建议客户尽快做好自查及防护。**
